## Getting Started

### Clone this project and access it directory

```
git clone https://github.com/Cheersupzoo/flutter_simple_isolate.git
cd flutter_simple_isolate
```

### Install dependency

```
flutter pub get
```

### Run the app

```
flutter run
```

- For smooth animation, recommend run in release/profile mode

### Testing

```
flutter test
```
